Objective

The primary objective of this project is to analyze and explore the sales data to identify key patterns, trends, and factors influencing sales performance. Additionally, we aim to assess the company's shipping efficiency by analyzing shipping time ranges and shipping costs per unit.

Dataset Description

The dataset consists of multiple columns, including 'Order ID', 'Order Date', 'Ship Date', 'Ship Mode', 'Customer ID', 'Customer Name', 'Segment', 'Country', 'City', 'Postal Code', 'Region', 'Product ID', 'Category', 'Sub-Category', 'Product Name', 'Sales', 'Quantity', 'Discount', 'Discount value', 'Profit', 'COGS', 'Shipping time range', 'Year sales', and 'Shipping cost per unit'. Each row represents a unique sales transaction made by the customers of the e-commerce company.
